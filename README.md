# Vextris

A block-stacking puzzle game built with Phaser.js.

Clear lines, stack blocks, survive the increasing speed. Classic Tetris-style gameplay with a custom visual style.

## Quick Start

```bash
npm install
npm run dev       # Development server
npm run build     # Production build
```

## Features

- Classic block-stacking mechanics
- Score tracking with benchmarks
- Phaser 3 game engine
- TypeScript
